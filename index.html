In this project I created a device that's called an Oximeter, it's used to measure and graph blood oxygen levels, which is dependent on the pulse. I used a simple DIY sensor to measure changes in blood oxygen levels, the sensor uses IR LED and a photo transistor to measure the change in blood oxygen levels.
Now here where things get more interesting, I have the data which is essentially the user's pulse, I could use the Arduino to measure the user's BPM directly, but I'll be limited by what the Arduino can do.
Instead, I used the Arduino to communicate the data stream to the computer, now I can collect the in MATLAB as it is being stream, I can use this powerful tool to graph the data or process it however I want.
