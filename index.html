High-Voltage Differential Sensor

This project is my graduation project, I worked on it with 3 other student in designing, implementing, and testing this sensor. It is a sensor that allows measurements of 1 KV DC or AC peak, it scales down the voltage either 100 time or 1000 time and isolates the user from the measured voltage.

This sensor can be very useful for hobbyists who want to work with inverters, smart grid networks or renewable energy applications, it has the advantage of being very affordable while providing a safe measurement.

Move the slider and take look inside:



The key component in this sensor is the AMC1200, it is an IC that's used in isolated current measurement, but with an appropriate voltage divider added to it can also measure voltage. Other than the IC and the DC-DC isolator; all components are off the shelf and inexpensive, the total cost of creating one sensor is only $36 including the PCB and the 3D printed case!

Graphing pulse with MATLAB

In this project I created a device that's called an Oximeter, it's used to measure and graph blood oxygen level which is dependent on the pulse. I used a simple DIY sensor to measure changes in blood oxygen levels, the sensor uses a common IR LED and a photo transistor to measure the change in blood oxygen levels.
Now here where things get more interesting, I have the data which is essentially the user's pulse, I could use the Arduino to measure the user's BPM directly, but I'll be limited by what the Arduino can do. Instead, I used the Arduino to communicate   

the data stream to the computer, now I can collect the in MATLAB as it is being stream, I can use this powerful tool to graph the data or process it however I want.

As you can see the graph is showing my pulse, the update speed is very good, I can see the graph rise right after feeling a heart beat.The resolution of the graph is 300 point x 1000 point, the horizontal axie acts like a buffer, this is useful so that the data is saved, I was able to use it to calculate the pulse from the graph continuously, but any process that we think of can be applied to received data stream to give us the information and measurements we need.

Wi-Fi controlled relay

Wirelessly controlled appliances are becoming very popular, devices such as Wi-Fi controlled lights or AC units are a good example. I really liked this idea and decided to create it, I did not want to create a specific appliance, so, I decided to create a Wi-Fi controlled relay.

The idea is to add this Wi-Fi relay to any device; to make it controllable by Wi-Wi. I went with the ESP8266 module since it is very popular and does not cost much, the project had two sides: hardware and software, the hardware part is about designing the circuit and the PCB, the software part is about writing the code for the page and the ESP8266 module as well.

The project took me about 4 weeks, the software side took most of the time, I am happy how the final product turned out, it is very small and easy to add to any device, which is exactly what I wanted!

The page for the project is very simple, just a standard HTML/CSS page. At first, the page would send data to the ESP8266 by requesting a specific address, this method worked but was very slow. After some research I found a better way which is to use JavaScript and send data using the 'Get Method' which improved the speed drastically.
The project turned out very well overall, I tested it by wiring a light bulb and using the relay to turned on and of remotely, and it worked as expected.

Fully DIY desktop CNC

In this project I created a CNC from scratch, the goal was to build a low-cost CNC and to used it for 3D printing or even for machining aluminum.

I started by designing the two carriages for X and Y axes, I kept everything simple, each carriage will have its own stepper motor (NEMA 17), and the Z axis will be suspended with its own stepper as well.



The most complex part of the project was creating linear motion from rotational motion, in commercial CNCs a lead screw is used, I used a threaded road, which is essentially the same, that only difference is the pitch of the threads. Threaded rods have finer threads making it harder to align, but I took it into account in my design and made all the critical parts fully adjustable.
I used the arduino and the DRV8825 motor controller to control the steppers, the softwere I used is a well known opencode software called GRBL which had all the futre of commercial CNCs in it.

I have tested it further by installing a motor with a milling bit on the Z axis, I engraved a piece of wood and it worked well, I have a lot of improvements to add to the project, I believe that with the added edits of what I have leaned by now; the designed can be even simplified more, making it cheaper and easier to build for beginners as well.
